=================================
ABCDeweb3.js - Ethereum JavaScript API
=================================

ABCDeweb3.js is a collection of libraries which allow you to interact with a local or remote ethereum node,
using a HTTP or IPC connection.

The following documentation will guide you through :ref:`installing and running ABCDeweb3.js <adding-ABCDeweb3>`,
as well as providing a :ref:`API reference documentation <#id1>` with examples.
